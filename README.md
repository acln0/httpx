httpx
=====

`import "acln.ro/httpx"`

[![GoDoc](https://godoc.org/acln.ro/httpx?status.svg)](https://godoc.org/acln.ro/httpx)

Convenience extensions to `net/http`. Not much to see here. You
probably don't want to use this anyway.
